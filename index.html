About

World Negative One is a user-friendly Direct Connect Hub Suite built on the PtokaX 0.5 platform. The suite builds around the file-sharing capability given by the serving platform to make the overall experience more fun and easy for the user. The suite has a plethora of features for both the host and the user like Automated backup, LAN game server scanners, Television series information retrieval, SSH logging and many more!

The hub suite project was started for local use in BITS-Pilani, Pilani Campus by its original developers in September, 2013. It was developed originally with the theme: Gaming and Art. Evidently, its primary target audience were LAN gamers and people with an artistic outlook. Since then, it has grown exponentially in terms of features and outreach. The suite was made open-source in January, 2014 with a Git repository hosted on Github. The hub is still in its infant stages of development with a lot of scope to improve and dream.

The suite is built on top of the PtokaX 0.5 Direct Connect Hub with hub scripts written in Lua 5 API 2. The suite uses Lua, Shell, Powershell and Java for implementing various modules. The suite is designed to be as cross-platform as possible. Lua is primarily used as the scripting language for the Ptokax interface. The directory structure helps define overall organization of modules and facilitates information flow between them. Extensive documentation of each module is in the Documentation directory of the main source tree.

PtokaX DC Hub is a multi - platform server application for Neo - Modus Direct Connect (NMDC) Peer-to-Peer sharing network. Started in May 2002 by Ptaczek, PtokaX has evolved into a mature, feature rich, stable and widely used server application. As the first of it's breed PtokaX utilizes ultra-fast crossplatform LUA scripting engine, designed by Tecgraf, the Computer Graphics Technology Group of PUC-Rio (the Pontifical Catholic University of Rio de Janeiro in Brazil).

- Nilay Binjola
